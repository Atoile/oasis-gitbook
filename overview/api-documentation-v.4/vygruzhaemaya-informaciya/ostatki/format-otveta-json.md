# Формат ответа JSON

```javascript
[
    {
        "article": "514209",
        "stock": 11,
        "stock-remote": 0,
        "price": "3835.00"
    },
    {
        "article": "11101902",
        "stock": 2200,
        "stock-remote": 0,
        "price": "249.00"
    },
    ...
]
```

> Расширить выгрузку можно перечислив поля в параметре **fields** \(id, stock, article, stock-remote, price, reserve\).
