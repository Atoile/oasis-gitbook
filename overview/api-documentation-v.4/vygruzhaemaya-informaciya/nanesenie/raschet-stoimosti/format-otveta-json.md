# Формат ответа JSON

```text
{
    "status": "ok",
    "data": {
        "client": 13.5,
        "dealer": 11.475
    }
}
```

* **status** - статус операции. ok - запрос отработал корректно, error - в процессе выполнения произошли ошибки
* **data** - данные по стоимости нанесения
  * client - клиентская цена
  * dealer - цена для дилера \(со скидкой\)
