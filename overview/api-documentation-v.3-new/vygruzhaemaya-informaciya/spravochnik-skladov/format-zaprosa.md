# Формат запроса

```text
https://api.oasiscatalog.com/v3/warehouses?format=xml
```

[**https://api.oasiscatalog.com/v3**](https://api.oasiscatalog.com/v3) ****- основной url

**warehouses** - экспорт складов

**format** - параметр, формат выгрузки \(xml или json\).
