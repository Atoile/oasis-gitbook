**min** - цена за минимально нанесение в 1 мм

**max** - максимальная цена за нанесения

**from** - минимальное количество в тираже, шт

**to** - максимальное количество в тираже, шт

**sum** - общая цена за тираж, не зависящая от количества

**price** - цена за одну штуку в копейках

**client** - стоимость нанесения для клиента

**dealer** - стоимость нанесения для дилера
